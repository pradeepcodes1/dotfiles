# Tools

Tools and applications configured in this dotfiles repo.

## Shell & Terminal

- **zsh** - Shell
- **tmux** - Terminal multiplexer
- **ghostty** - Terminal emulator
- **atuin** - Shell history search

## Editor

- **neovim** - Text editor
- **lazy.nvim** - Plugin manager

## Navigation & Files

- **yazi** - Terminal file manager
- **eza** - Modern ls replacement
- **zoxide** - Smart cd

## Version Management

- **mise** - Runtime version manager (go, node, python)
- **homebrew** - Package manager
- **nix** - Package manager (for temporary shells)
- **chezmoi** - Dotfiles manager

## macOS Window Management

- **aerospace** - Tiling window manager
- **karabiner-elements** - Keyboard customization

## Other

- **pass** - Password manager (for secrets in templates)
- **restic** - Backup tool
- **bitwarden** - Password manager (GUI)
- **localsend** - Local file sharing
